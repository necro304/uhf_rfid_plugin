## 1.0.0

* Initial release
* UHF RFID Features:
  - Initialize/close UHF reader
  - Continuous and single inventory scanning
  - Read/write tag data (EPC, TID, USER, RESERVED memory banks)
  - Power configuration (5-33 dBm)
  - Frequency region settings (USA, EU, China, Korea)
  - Inventory filtering
  - Tag lock/kill operations
  - Physical trigger button support
  - Temperature monitoring
* Barcode/QR Scanner Features:
  - Initialize/close barcode scanner
  - Scan barcodes and QR codes
  - Configurable timeout (1-10 seconds)
  - Scan mode selection (Broadcast or Focus Input)
  - Real-time scan results via stream
